# ArmorDetector - 装甲板检测系统

基于OpenCV的实时装甲板检测系统

## 功能特点
- 实时装甲板检测
- 灯条识别和几何筛选（面积、角度、宽高比）
- 数字识别(暂时还未添加)
- 实时处理

## 环境要求
- OpenCV 4.10.0
- g++ 编译器

## 编译方法
ubuntu22.04
```bash
g++ -std=c++11 main.cpp -o  main onfig --cflags --libs opencv4`

## 运行程序
```bash
./main
```

## 检测流程
1. 图像预处理（灰度化、二值化）
2. 轮廓查找
3. 几何筛选（面积 → 宽高比 → 角度）
4. 灯条配对
5. 装甲板识别

## 技术细节
- 使用旋转矩形(minAreaRect)进行几何特征提取
- 多条件筛选：面积(300-4000)、宽高比(2.0-15.0)、角度(60-120度)
- 基于轮廓分析的目标检测
